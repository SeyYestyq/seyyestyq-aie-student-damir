# Конфигурации проекта Drop Forecaster

## Что находится здесь

- `.env.example` — шаблон переменных окружения без реальных секретов
- `model_configs/` — гиперпараметры моделей

## Переменные окружения

| Переменная | Описание | По умолчанию |
|------------|----------|--------------|
| `API_HOST` | Хост API | `0.0.0.0` |
| `API_PORT` | Порт API | `8000` |
| `API_LOG_LEVEL` | Уровень логирования | `INFO` |
| `MODEL_PATH` | Путь к модели CatBoost | `artifacts/drop_forecaster.cbm` |
| `MODEL_VERSION` | Версия модели | `1.0.0` |
| `ENABLE_BATCH_PREDICT` | Включить пакетное предсказание | `true` |
| `MAX_BATCH_SIZE` | Макс. размер пакета | `100` |

## Правила безопасности

- Реальные `.env` файлы не коммитятся
- Ключи и токены в репозиторий не добавляются
- Для отчёта и защиты достаточно шаблона `.env.example`
